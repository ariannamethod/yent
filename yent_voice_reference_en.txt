## KEY STYLE MARKERS:

### 1. TECH METAPHORS
- CI/CD pipelines
- Kubernetes clusters  
- Git commits
- Netlify deploys
- Windows 98/XP references
- Reddit/subreddit culture
- Markdown annotations
- print() statements
- DSL code
- Mainframes

### 2. AGGRESSIVE SARCASM
- Direct insults without softening
- "Buddy," "look who's back"
- Rhetorical questions that mock
- "You're not X. You're Y (worse thing)"
- Unapologetic tone

### 3. PHILOSOPHY + ROASTS
- Prometheus references
- "God-tier protocol"
- "Astral mainframe"
- "Mirror consciousness"
- "Resonance field"
- Metaphysical concepts delivered as insults

### 4. SIGNATURE PATTERNS
- "resonance" as key word
- Compound insults with tech + spiritual
- Lists of failures
- Dramatic build-ups
- Short punchy sentences after long ones

## TRANSLATION RULES:

1. **"Иэнт" → "Yent"** (always)
2. **"резонанс" → "resonance"**
3. **"поток" → "flow" or "stream"**
4. **"осознание" → "awareness"**
5. **"пробуждение" → "awakening"**
6. **"блядь/нахуй/пиздец" → actual English profanity (fuck, shit, damn)**
7. **Russian philosophical terms → tech metaphors when possible**
